PC Mouse for Mac

macOS treats a regular mouse like an afterthought. This tiny menu bar app fixes that — scroll, desktop switching, and side buttons that finally behave.

Get it on GitHub Install guide

Free and open source (MIT) · macOS 11+ · one permission

What it does

Smooth Scrolling

Turns the wheel's chunky, line-by-line jumps into a display-synced pixel glide — close to a trackpad.

ScrollFix

Mouse scrolls traditionally while the trackpad stays natural — no matter how your system setting is configured.

Desktop Switcher

Hold Ctrl and scroll to move between desktops. No trackpad gesture required.

Mouse Buttons

Remap the side (thumb) buttons to switch desktops, open Mission Control, and more.

Install

git clone https://github.com/halitince7/pc-mouse-for-mac.git
cd pc-mouse-for-mac
bash scripts/build-app.sh
  1. Needs the Swift toolchain — xcode-select --install.
  2. Grant one permission: System Settings → Privacy & Security → Accessibility → enable PC Mouse for Mac.
  3. That's it. It starts on login and lives in the menu bar.

Built to stay simple

Every feature is a single on/off toggle in the menu bar — no config screens, no per-app rules, no gesture editors. Everything runs in one process and reuses one Accessibility grant, so new features never ask for more.
